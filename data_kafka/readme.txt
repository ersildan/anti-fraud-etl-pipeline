Папка со скриптами producer и consumer для кафки.
producer_to_kafka => запускает генераторы и загружает данные в кафку.
consumer_from_kafka => запускает чтение топиков с кафки и загружает данные в hdfs.

Folder with producer and consumer scripts for kafka.
producer_to_kafka => starts generators and uploads data to kafka.
consumer_from_kafka => starts reading topics from kafka and uploads data to hdfs.
